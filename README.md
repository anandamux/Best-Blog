# Best Blog Site 💜

Um sistema de blog moderno construído com Node.js, Express e EJS. O projeto apresenta um design responsivo com efeito "Glassmorphism" (Vidro Fosco).

## 🚀 Tecnologias

- **Node.js** & **Express** (Backend)
- **EJS** (Template Engine)
- **CSS3** (Flexbox, Grid, Animações e Glassmorphism)

## 📸 Funcionalidades

- Visualização de Posts
- Criação de novos Posts (armazenamento em memória)
- Página "Sobre Nós" e "Contato"
- Design responsivo para Mobile e Desktop

## 📦 Como rodar

1. Clone o repositório
2. Instale as dependências: `npm install`
3. Rode o servidor: `node solution.js` (ou `index.js`)
4. Acesse: `http://localhost:3000`
